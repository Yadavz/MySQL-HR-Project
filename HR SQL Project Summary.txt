This HR project involves the analysis of HR data for a company using SQL queries on a MySQL database. The project includes creating a new database, renaming columns, checking table structure, cleaning and formatting data, adding a new column, using aggregate functions, grouping and sorting data, calculating turnover rate, and analyzing employee count over time. The analysis provides valuable insights into HR data and showcases the use of SQL queries to manipulate and analyze data, which can be used to make informed decisions regarding the company's HR policies.
